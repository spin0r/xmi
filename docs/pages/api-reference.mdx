# API Reference

The API is exposed on the configured port (default `3000`).

## 1. Process Pastebin URL

Downloads images from a pastebin, uploads them to `imx.to` (optionally inside a new gallery), and returns direct URLs.

**Endpoint:** `POST /api/upload-pb`

**Headers:**
- `Content-Type: application/json`

**Body:**
```json
{
  "pbUrl": "https://pb.dotrhelvetican.workers.dev/abcd",
  "galleryName": "Optional Gallery Name"
}
```

**Response (Success):**
```json
{
  "success": true,
  "total_found": 15,
  "successful_uploads": 15,
  "direct_urls_extracted": 15,
  "gallery_url": "https://imx.to/g/gallery123",
  "pb_url": "https://pb.dotrhelvetican.workers.dev/xyz",
  "direct_urls": [
    "https://imx.to/u/a/image1.jpg",
    "https://imx.to/u/a/image2.jpg"
  ]
}
```

---

## 2. Extract IMX Links

Given an array of viewer links, this endpoint resolves them to their raw, direct `img` links.

**Endpoint:** `POST /api/extract-imx`

**Headers:**
- `Content-Type: application/json`

**Body:**
```json
{
  "links": [
    "https://imx.to/i/abc",
    "https://imx.to/i/xyz"
  ]
}
```

**Response:**
```json
{
  "success": true,
  "total": 2,
  "extracted": 2,
  "direct_urls": [
    "https://imx.to/u/a/abc.jpg",
    "https://imx.to/u/a/xyz.jpg"
  ]
}
```

---

## 3. Upload File

Directly upload an image to `imx.to` using `multipart/form-data`.

**Endpoint:** `POST /api/upload-file`

**Headers:**
- `Content-Type: multipart/form-data`

**Body:**
- `image`: The file payload.

**Response:**
```json
{
  "success": true,
  "image_url": "https://imx.to/u/a/abc.jpg",
  "thumbnail_url": "https://imx.to/u/t/abc.jpg",
  "gallery_id": null
}
```

---

## 4. Health Check

Returns the current uptime and server status. Perfect for pointing UptimeRobot to keep Render services awake.

**Endpoint:** "GET /health"

**Response:**
```json
{
  "status": "ok",
  "uptime": 123.45,
  "timestamp": "2024-01-01T00:00:00.000Z"
}
```
