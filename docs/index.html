IMX Uploader API

Express API for uploading images to imx.to, processing Pastebin URLs, and extracting direct links.

POST /api/upload-pb

Downloads images from a pastebin URL, uploads them to imx.to (optionally in a new gallery), and returns direct URLs.

{
  "pbUrl": "https://pb.example.com/abc",
  "galleryName": "Optional Gallery Name"  // optional
}

Response:

{
  "success": true,
  "total_found": 15,
  "successful_uploads": 15,
  "direct_urls_extracted": 15,
  "gallery_url": "https://imx.to/g/gallery123",
  "pb_url": "https://pb.example.com/xyz",
  "direct_urls": ["https://imx.to/u/a/image1.jpg", "..."]
}

POST /api/extract-imx

Resolves imx.to viewer links to raw direct image URLs.

{ "links": ["https://imx.to/i/abc", "https://imx.to/i/xyz"] }

Response:

{
  "success": true,
  "total": 2,
  "extracted": 2,
  "direct_urls": ["https://imx.to/u/a/abc.jpg", "https://imx.to/u/a/xyz.jpg"]
}

POST /api/upload-file

Upload an image file directly via multipart/form-data. Field name: image.

Response:

{
  "success": true,
  "image_url": "https://imx.to/u/a/abc.jpg",
  "thumbnail_url": "https://imx.to/u/t/abc.jpg",
  "gallery_id": null
}

GET /health

Returns server status and uptime. Use with an uptime monitor to keep Render free-tier awake.

{ "status": "ok", "uptime": 123.45, "timestamp": "2024-01-01T00:00:00.000Z" }
